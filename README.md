##  Project Overview
This is an interactive web-based **number guessing game** built using HTML, CSS, and JavaScript. The game uses a simple binary logic trick to guess a number between **1 and 63** based on user selections.

---

##  How It Works
- The user thinks of a number from **1 to 63**
- Six colored cards display different number sets
- Each card represents a binary value:
  - 1, 2, 4, 8, 16, 32
- The user selects all cards containing their number
- The program adds selected values together
- The final sum reveals the chosen number 🎉

---

##  Features
- Color-coded number cards with rounded styling
- Binary-based number guessing logic
-  Interactive checkbox selection
- Instant result display using JavaScript
- Navigation bar linking multiple project pages

---

## Technologies Used
- **HTML5** – page structure
- **CSS3** – styling and layout (rounded cards, colors)
- **JavaScript** – logic and interactivity

---

## JavaScript Logic
The core function loops through selected checkboxes, adds their values, and displays the result:

- Collects checkbox inputs
- Checks which are selected
- Sums their values
- Outputs final number to the page
